# 📄 Document Q&A

基于文档的智能问答系统，使用 Streamlit + OpenAI GPT-3.5 实现。

## 功能特性

- 支持上传 PDF、txt 等文档
- 基于文档内容进行智能问答
- 使用 OpenAI GPT-3.5 模型
- 简洁的 Web 界面

## 技术栈

- Python
- Streamlit
- OpenAI GPT-3.5
- LangChain

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

设置 OpenAI API Key：

```bash
export OPENAI_API_KEY="your-api-key"
```

### 3. 运行应用

```bash
streamlit run streamlit_app.py
```
